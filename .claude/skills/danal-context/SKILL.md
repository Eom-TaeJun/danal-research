## 핵심 제품 컨텍스트

### KRW 스테이블코인 SaaS
- 원화 연동 스테이블코인 발행·유통·결제·정산 통합 플랫폼
- B2B SaaS 모델 — 기업이 직접 스테이블코인 발행 가능
- 경쟁 우위: 기존 결제 인프라 + 블록체인 결합

### x402 프로토콜
- AI 에이전트 자동 결제 인프라
- HTTP 402 기반 — AI가 콘텐츠/서비스 비용을 자동 처리
- 스테이블코인 결제 레이어로 활용

### 페이코인 (PCI)
- 편의점 4사 (CU·GS25·세븐일레븐·이마트24) 실물 결제 연동
- 블록체인 기반 포인트·결제 통합

## 분석 시 연결 원칙

모든 외부 분석(시장, 규제, 경쟁사)을 다날 3축에 연결:

```
GENIUS Act 통과
→ 달러 스테이블코인 규제 명확화
→ 원화 스테이블코인 SaaS 사업 확장 기회 (한국판 규제 선례)
→ 다날 KRW 스테이블코인 타이밍 가속 가능성
```

```
Fed 금리 인하
→ 스테이블코인 준비금 이자 수익 감소 (Circle, Tether 부정적)
→ 반면 결제 스테이블코인 채택 증가 (수수료 모델 유리)
→ 다날 SaaS 수수료 모델은 금리 중립
```
