## 섹션 4. Market Opportunity — TAM/SAM/SOM

**TAM 추정 방법**:
```
Top-down: 글로벌 결제 시장 × 디지털자산 침투율 예상 (출처 + 연도 명시)
Bottom-up: 현재 addressable 가맹점 수 × ARPU
```

**SOM 기준 (스테이블코인 기업)**:
- SOM = 현재 발행량 또는 거래 볼륨 (CoinGecko, 날짜 명시)
- 성장 전제가 있으면 연도 명시 필수

**경쟁사 비교표 (최소 4개)**:
| 경쟁사 | 강점 | 약점 | 시장점유율 | 출처 |
|--------|------|------|----------|------|

## 섹션 5. 재무 실적 — 수치 품질 기준

**최소 요구사항**:
- 연도별 매출·순이익·마진율 (최소 2개년)
- YoY 성장률 명시
- 추정치(E)는 출처 또는 "추정" 명시

**차트 임베드**:
```markdown
![Revenue Trend](../charts/revenue_trend_YYYYMMDD.png)
```

**수치 허용 기준**:
- Bloomberg, 공식 투자자료, SEC 공시: 그대로 사용
- Perplexity 추출: "추정" 또는 "확인 필요" 주석 필수
- 인터넷 수집 불가: "데이터 없음 — 공시 확인 필요" (수치 추정 금지)

## 섹션 6. Investment Thesis — Bull/Bear 균형

**형식 (必 지킬 것)**:
```markdown
**Bull Case:**
1. [구체적 근거 — with 수치]
2. [구체적 근거]
3. [구체적 근거]

**Bear Case:**
1. [리스크 — with 조건]
2. [리스크]
3. [리스크]
```

**품질 기준**:
- "성장 가능성이 있다" ❌ → "GENIUS Act 수혜 → 기관 USDC 채택 가속, 2025 YoY +114% 확인" ✅
- Bull만 3개, Bear 없음 ❌ (균형 필수)
- Bear를 "단점"으로 표현 ❌ → "조건부 시나리오"로 표현 ✅

## 섹션 7. 밸류에이션 — 피어 비교

**피어 선정 기준**:
- 동일 섹터 + 상장 기업 우선 (비상장은 추정치 주석)
- 최소 2개 멀티플 (EV/Revenue, EV/EBITDA, P/E 중 선택)

**멀티플 해석 기준**:
```
EV/Revenue > 피어 평균: 프리미엄 판단 → 성장률로 정당화 필요
EV/Revenue < 피어 평균: 디스카운트 → 리스크 또는 저평가 판단
```

**다날 연결 필수**:
- "이 밸류에이션이 다날의 파트너십/투자 여부에 어떤 의미인가"

## 섹션 8. Key Risks — 리스크 강도 표

**4대 리스크 카테고리**:
| 카테고리 | 예시 | 강도 (⭐~⭐⭐⭐) |
|---------|------|-----------|
| 규제 | CBDC 도입, 스테이블코인 금지 | ⭐⭐⭐ |
| 시장 | 금리 역전, 경쟁사 점유율 | ⭐⭐ |
| 운영 | 해킹, 청산 실패 | ⭐⭐ |
| 재무 | 준비금 부족, 디페그 | ⭐⭐⭐ |

**강도 기준**:
- ⭐⭐⭐: 기업 존속을 위협하는 리스크
- ⭐⭐: 수익성에 영향을 주는 리스크
- ⭐: 모니터링 필요하나 현재 미미
