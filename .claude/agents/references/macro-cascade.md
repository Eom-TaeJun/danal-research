# 거시 전파 분석 (Cascade Transmission) — 참조 문서

> macro-analyst Step 6에서 호출. IM 요청 시에만 실행.
> 모든 α·β·γ 값은 실시간 데이터·뉴스에서 도출. 하드코딩 금지.

## 데이터 수집 (WebSearch)

```
KOSPI·KOSDAQ 현재 레벨 + WoW 변화율
외국인 순매수/순매도 최근 5거래일 합계
한국 정부 재정기조 (최근 예산·금통위 결정)
글로벌 리스크 뉴스 (미국 금융 시스템, 지정학)
```

## α 도출 (글로벌 → 한국)

이론: 신흥국 자본흐름 — "flight to quality"(동조) vs "search for yield"(반사이익)를 외국인 순매수 방향으로 확인.

| 수집 신호 | 경로 | α |
|---------|:----:|:---:|
| 외국인 순매도 + KRW 약세 | 동조 하락 | 0.6~0.9 |
| 외국인 순매수 + KRW 강세 | 반사이익 | 0.2~0.4 |
| 확장재정 + 한은 인하 기조 | 국내 독립 | 0.1~0.3 |

## β 도출 (한국 → 섹터)

이론: 유동성 사이클 — M2 확대 + 저금리 시 위험선호가 성장·테마주로 집중.

| 수집 신호 | 경로 | β |
|---------|:----:|:---:|
| KOSDAQ > KOSPI 초과성과 + 테마 활성 | 유동성 장세 | 0.7~0.9 |
| 규제 불확실 또는 섹터 비활성 | 위축 | 0.2~0.4 |

## γ 도출 (섹터 → 기업)

이론: 매출 지역 집중도 — 국내 비중이 높을수록 섹터 신호가 실적에 직접 반영.

| 기업 특성 | γ |
|---------|:---:|
| 국내 매출 90%+ (순수 내수) | 0.8~0.9 |
| 국내 사업 有 + 글로벌 자산 | 0.3~0.6 |
| 글로벌 자산 + 한국 간접 파트너 | 0.2~0.4 |
