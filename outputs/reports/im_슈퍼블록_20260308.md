# Investment Memorandum — 슈퍼블록 (Superblock)
> 작성일: 2026-03-08 | 초안 (Draft)

## 1. Executive Summary

슈퍼블록은 경량화 L1 블록체인 '오버 프로토콜(Over Protocol)'을 개발·운영하는 한국 블록체인 인프라 기업이다. 서울대 블록체인 학회 '디사이퍼' 출신의 김재윤 대표가 창업했으며, SK·넷마블·네이버Z 등으로부터 시리즈A 약 100억원을 조달했다. 다날은 2025년 12월 전략적 투자를 통해 오버 프로토콜의 스테이블코인 정산 인프라를 KRW SaaS·x402 AI 결제 파이프라인에 내재화하는 기술 파트너로 낙점했다.

**투자 의견:** ☐ 관심  ☑ 검토  ☐ 보류

## 2. Company Overview

**사업 모델:** 오버 프로토콜 네이티브 토큰 이코노미(네트워크 수수료·스테이킹) + B2B 스테이블코인 정산 API SaaS. PG·핀테크·가맹점에 스테이블코인 결제/정산 API, 지갑 인프라, 멀티체인 라우팅 SDK를 제공하고 월 사용료·거래당 수수료·엔터프라이즈 라이선스로 수익화한다.

**핵심 제품/서비스:**
- 오버 프로토콜 (Over Protocol) — 경량화 L1 블록체인 (Ethanos 기술 기반)
- 오버 월렛 (Over Wallet) — 개인·기업용 자산 관리 지갑
- 오버 노드 (Over Node) — 개인 PC 기반 노드 운영·보상 시스템
- 스테이블코인 정산 오케스트레이션 API (다날 협력 개발 중)

## 4. Market Opportunity

**시장 규모:** 글로벌 L1 블록체인 플랫폼 시장(수십억 달러 규모)을 1차 타깃으로 하며, 국내 스테이블코인 정산·커스터디·지갑·토큰화 백엔드를 합친 인접 시장은 연 2,000억~5,000억원으로 추정된다. 스테이블코인 정산이 제도권 PG에 편입되면 결제 소프트웨어 시장 일부를 잠식 가능.

**경쟁사 (2개 레이어):**
- **L1 경쟁**: 카이아(Kaia), 위믹스(WEMIX), 아이콘(ICON/파라메타)
- **인프라·미들웨어 경쟁**: 람다256(Luniverse), 헥슬란트(Hexlant), 해치랩스(HAECHI LABS)

## 6. Investment Thesis

**Bull Case:**
- 독자적인 경량 노드 기술(Ethanos)으로 탈중앙성·확장성 확보 — 자본집약적 L1 경쟁 대비 차별화
- 다날(국내 1위 PG)과의 독점 기술 파트너십 → 실사용 사례(KRW SaaS·x402) 확보로 네트워크 효과 조기 창출
- 한국 PG 문맥에서 가맹점 온보딩·원화 정산·규제 로깅·AML 연동까지 묶으면 로컬 전환비용이 높아짐

**Bear Case:**
- 이더리움·솔라나 등 글로벌 L1과의 경쟁 — 개발자·사용자 생태계 확보 불확실성
- x402는 오픈 규격에 가까워 대형 거래소·글로벌 결제 API 사업자의 빠른 복제 가능
- 스테이블코인 정산 인프라의 운영 복잡도 (체인 finality·reorg·issuer freeze·RPC 장애 등 흡수 필요)
- 현재 뚜렷한 매출 없이 투자금 의존 — 지속 가능한 BM 증명 과제

## 8. Key Risks

- **보안 리스크**: 키관리 실패·스마트컨트랙트 취약점·브리지 해킹이 정산 시스템 신뢰 훼손
- **자산 리스크**: 스테이블코인 issuer 동결권한·디페깅·블랙리스트가 실시간 결제 UX 붕괴
- **규제 리스크**: 국내 디지털자산기본법 시행령 확정 지연 시 실제 결제 볼륨 창출 지연
- **시스템 통합 리스크**: 기존 PG 원장과 온체인 원장 불일치·취소환불 처리·지연 확정성 문제

## 10. 다날 전략 연결

**투자 배경**: 다날은 2025년 12월 슈퍼블록에 전략적 투자를 단행. PG의 약점(온체인 자산 처리 레이어)을 슈퍼블록 기술로 보완하고, 다날핀테크·페이프로토콜과 결합해 스테이블코인 수납·크로스보더 정산·API 기반 B2B 결제를 단계적으로 제품화하는 구도.

**시너지 경로**:
- **단기**: 오버 프로토콜 위 스테이블코인 API → 다날핀테크 KRW SaaS 정산 레이어 연동
- **중기**: x402 프로토콜 → 다날 AI 에이전트 결제 (슈퍼블록 미들웨어 기반)
- **장기**: 페이프로토콜 + 슈퍼블록 → 크로스보더 스테이블코인 정산망

**모니터링 지표**:
- [ ] 슈퍼블록 API 다날핀테크 연동 시점 및 거래량
- [ ] x402 미들웨어 상용화 거래량
- [ ] 원화 스테이블코인 규제 확정 후 KSC 정산 전환 타임라인
- [ ] 오버 프로토콜 메인넷 TVL 및 생태계 확장 속도

**리스크 모니터링**:
- x402 오픈 규격 → 글로벌 플레이어 복제 속도 추적
- 스테이블코인 issuer 동결/디페깅 리스크
- 개발자 생태계 성장 속도 (L1 경쟁력 유지 여부)
