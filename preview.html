분석 파이프라인

  사용자 요청
      │
      ├── --brief   ──→  collect.py  (FRED + CoinGecko + Perplexity 뉴스)
      │                      └→  report.py  →  brief_YYYYMMDD.md
      │
      ├── --im "기업명"  ──→  research.py  (Perplexity 기업 리서치)
      │                          └→  report.py  →  im_기업명_YYYYMMDD.md
      │
      ├── --screen  ──→  collect.py → analyze.py → chart.py → report.py
      │                  레짐 판단 (수익률 곡선 + Fed 스탠스)
      │                  다날 비즈니스 함의 자동 도출
      │                      └→  screen_stablecoin_YYYYMMDD.md + 3종 차트
      │
      └── --analyze ──→  analyze.py  →  analysis_YYYYMMDD.json
                         레짐 + 스테이블코인 시그널 + 함의

주요 기능

python main.py --brief
주간 브리핑
거시경제 스냅샷 + 스테이블코인 시장 + 핵심 뉴스 3건 + 투자 시사점
python main.py --im "Circle"
투자 검토 보고서 (IM)
10섹션 구조: Executive Summary ~ 다음 단계. 재무·밸류에이션·경영진 포함
python main.py --screen stablecoin
섹터 스크리닝
레짐 판단 + 시장 현황 + 다날 비즈니스 함의 + 기회·리스크 매트릭스
python main.py --analyze

Goldilocks / Overheating / Stagflation / Recession 판단. 다날 3축 함의 도출
