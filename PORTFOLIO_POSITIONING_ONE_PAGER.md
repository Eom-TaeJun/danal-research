# PORTFOLIO POSITIONING ONE-PAGER

기준일: 2026-03-21

이 문서는 `danal-research`를 다날 금융·투자 분석 인턴 포지션에 맞게 설명하기 위한 1페이지 기준안이다.

## 1. 목표 포지셔닝

`danal-research`는

**다날의 결제 플랫폼 확장 문맥을 이해한 상태에서 시장·기업·거시 자료를 내부 문서형 산출물로 바꾸는 지원자**

로 읽혀야 한다.

핵심은 `금융 관심`이나 `AI 활용`이 아니라 아래 4가지다.

- 시장 동향을 `brief`로 압축할 수 있는가
- 기업 검토 자료를 `IM 초안`으로 구조화할 수 있는가
- 거시/금융 데이터를 `chart`와 `scenario table`로 정리할 수 있는가
- 결과를 다날의 결제 본업, 가맹점 가치, 글로벌/외국인 결제 문맥에 연결할 수 있는가

## 2. 다날이 실제로 원하는 것

- `투자 리포트`, `IM 초안`, `금융 및 거시경제 데이터 분석`, `전략 보고 및 시각화`
- `통합 결제 플랫폼` 관점
- `가맹점 중심` 실행 언어
- `글로벌/외국인 결제`와 `디지털자산 확장`을 함께 읽는 태도
- 좋은 아이디어보다 `내부에서 바로 읽을 수 있는 문서` 생산 능력

참조:
- [OFFICIAL_SIGNALS_20260320.md](OFFICIAL_SIGNALS_20260320.md)
- [JOB_SIGNAL_ALIGNMENT.md](JOB_SIGNAL_ALIGNMENT.md)

## 3. 이 포트폴리오가 증명해야 하는 것

- 다날 JD를 `brief -> IM -> screen -> chart` workflow로 실제 구현했다
- 영어 자료, 규제 자료, 해외 기업 자료를 읽고 사업 의미로 번역할 수 있다
- 결론만 쓰지 않고 `확신도`, `반론`, `반증 가능 신호`, `watch point`를 남긴다
- 최종 투자 판단은 시니어가 한다는 전제 아래, 초안 작성과 자료 정리에 강하다
- harness 구조를 활용해 반복 실행 안정성과 품질 게이트를 유지한다

## 4. 덜 말해야 하는 것

- `AI를 잘 썼다`
- `투자팀 현직자가 하는 일을 그대로 했다`
- `VC/PE처럼 심사했다`
- `스테이블코인이 유망해서 만들었다`
- `멀티 에이전트가 핵심이다`

위 표현은 직무 적합성을 흐리거나 과장으로 읽힐 수 있다.

## 5. 바깥에 보이는 설명 순서

1. 다날 공고의 4개 핵심 업무
2. 대표 산출물 2~3개
3. 결제 본업 -> 가맹점 -> 글로벌/외국인 결제 -> 디지털자산 확장 연결
4. 확신도, watch point, 시니어 검토 전제
5. 마지막에 harness와 품질 게이트

즉 `구조`는 남기되 `산출물`이 먼저 보이게 설명해야 한다.

## 6. 합격권처럼 보이는 기준

- README 첫 화면만 봐도 `금융·투자 분석 인턴용 포트폴리오`로 읽힌다
- 첫 30초 안에 `brief`, `IM`, `chart`, `scenario`가 보인다
- 다날 특유의 `결제 플랫폼`, `가맹점`, `외국인 결제` 언어가 확인된다
- 스테이블코인은 단독 테제가 아니라 `결제 네트워크 확장 수단`으로 보인다
- harness는 남아 있지만 전면 주인공이 아니라 품질 보조 장치로 보인다
